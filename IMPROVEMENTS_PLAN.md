# План улучшений сайта ArtRichBot на основе знаний специалистов

## 1. КОПИРАЙТИНГ И ПСИХОЛОГИЯ УБЕЖДЕНИЯ

### Текущее состояние:
- Hero заголовок: "Чат-боты, которые" (неполный)
- Недостаточно эмоциональных триггеров
- Нет четкого USP (Unique Selling Proposition)

### Улучшения:
- ✅ Переписать Hero заголовок с использованием AIDA модели
- ✅ Добавить социальное доказательство (количество клиентов, отзывы)
- ✅ Использовать power words: "Революционный", "Гарантированный", "Эксклюзивный"
- ✅ Добавить срочность: "Ограниченное предложение", "Только для первых 100 клиентов"
- ✅ Переписать все CTA кнопки с глаголами действия

### Реализация:
```
Было: "Чат-боты, которые"
Стало: "Увеличьте продажи на 300% за 7 дней с AI-ботом, который работает 24/7"
```
